### scanf()

功能：scanf() 用于从标准输入（通常是键盘）中读取格式化的数据。

原型：int scanf(const char *format, ...)

### 参数：
- format：格式控制字符串，用来指定输入的数据类型。格式符类似于 printf()，如 %d（整型），%f（浮点型），%s（字符串）等。
  其他参数是指向变量的指针，用来存储从输入中读取的数据。

### 示例：
int a;
float b;
scanf("%d %f", &a, &b); // 读取一个整数和一个浮点数

返回值：返回成功读取的项数（即成功匹配的格式说明符的个数），如果出现输入错误或达到文件结束，则返回负值。

### 注意事项：
- scanf 可能会跳过空白符（空格、换行、制表符）等，而不处理输入缓冲区中残留的数据。
- scanf 的输入非常严格，如果输入的格式与预期的格式不匹配，可能会导致读取失败。

### getchar()

功能：getchar() 用于从标准输入读取一个字符。

原型：int getchar(void)

参数：无参数。

示例：
char ch;
ch = getchar(); // 读取单个字符

返回值：返回读取的字符（以 int 形式返回）。如果到达文件结束或发生错误，则返回 EOF（通常定义为 -1）。

### 注意事项：
- getchar() 读取的是缓冲区的第一个字符，因此它可能会阻塞等待用户输入，直到输入换行符后才能读取下一个字符。
- 通常需要与循环结合起来逐字符读取输入。

# fgets()

**功能**: 用于从输入流读取字符串的函数。相对于 `gets()`, `fgets()` 更安全，因为它允许指定读取的最大字符数，从而避免了缓冲区溢出的问题。因此，`fgets()` 是推荐使用的字符串输入函数。

**原型**: char *fgets(char *str, int n, FILE *stream);

## 参数:
- **str**: 指向字符数组的指针，用于存储读取到的字符串。
- **n**: 要读取的最大字符数（包括终止符 \0）。函数最多读取 n-1 个字符，最后一个位置留给字符串终止符 \0。
- **stream**: 输入流。通常为标准输入 stdin，但可以是任何文件流。

## 返回值:
- 返回值是指向 str 的指针，表示成功读取的数据。
- 如果在读取过程中遇到文件结束符（EOF）或发生错误，返回 NULL。

## 注意事项:
- `fgets()` 会在读取到换行符、读取到 n-1 个字符，或者遇到文件结束符（EOF）时停止读取。
- 换行符（\n）会被作为字符串的一部分存储在 str 中（如果读取到的话），并在字符串末尾自动添加终止符 \0。
## 二 输出函数
### printf()

功能：printf() 用于向标准输出（通常是屏幕）打印格式化的文本。

原型：int printf(const char *format, ...)

### 参数：
- format：格式控制字符串，指定如何格式化输出。常见的格式符有 %d（整型）、%f（浮点数）、%s（字符串等）。
- 其他参数是需要打印的变量。

### 示例：
int a = 5;
float b = 3.14;
printf("a = %d, b = %.2f\n", a, b); // 输出 a = 5, b = 3.14

返回值：返回成功打印的字符数。如果输出发生错误，返回负值。

### 注意事项：
- printf 非常灵活，可以打印各种类型的数据。
- 需要确保格式控制和变量类型匹配，否则会引发不正确的输出。
### putchar()

功能：putchar() 用于向标准输出打印一个字符。

原型：int putchar(int char)

参数：
- char：要打印的字符（传递的字符以 int 形式表示，但其值应是可表示为 unsigned char 的整数）。

示例：
- putchar('A'); // 输出字符 'A'

返回值：返回输出的字符（作为 unsigned char 类型转换为 int 返回）。如果发生错误，返回 EOF。

注意事项：
- putchar() 常用于逐个字符的输出，例如打印字符数组中的内容。
### puts()

功能：puts() 用于向标准输出打印一个字符串，并在输出后自动添加换行符。

原型：int puts(const char *str)

参数：
    str：要打印的字符串指针。

示例：
    puts("Hello, world!"); // 输出字符串并换行

返回值：成功时返回一个非负值，失败时返回 EOF。

#### 注意事项：
- puts() 在打印结束后自动添加换行符，不需要手动加 \n。
- 适合用于简单的字符串输出。
## 三 常量
### 字面常量
#### 1整数常量
前面放进制，后面放修饰符（有无符号，是否是长整型）
例如：
```c
int a= 42;
int b=052;
int c=0x2A;
unsigned int d= 42U;
long int e = 42L;

```
#### 2#define和const的区别
`#define` 是预处理指令，在编译前进行简单的文本替换，不分配内存、无类型检查、调试困难；而 `const` 是关键字，在编译阶段定义一个有类型的只读变量，分配内存、受作用域限制且类型安全。因此现代C编程中更推荐使用 `const` 定义常量，`#define` 则多用于条件编译或简单代码片段。

#### 3常量的作用
1 定义不可以变的值
2 提高代码的可读性
3 便于维护和修改

#### 4常量在嵌入式的应用
在单片机编程中，常量通常用于定义一些固定的参数，例如时钟分频值、波特率、定时器频率，硬件寄存器地址，引脚配置，状态和错误码，协议固定值，中断信号等。这些参数在系统运行时不会改变，使用常量来定义可以增加代码的清晰度和可维护性。


## 四 运算符
### 逻辑运算符短路特性
如果左侧就已经可以判断结果，那么右侧的表达式不会被执行
例如
```c
int a=1;
a==0&&a++
//这里a不会在自增
```
### 位移运算符
左移运算
将操作数的位向左移位，右侧补0
右移运算
如果是正数，将操作数的位向右移位，左侧补0
如果是负数，将操作数的位向右移位，左侧补1（符号位）

# switch 结构

switch 语句用于处理基于某个表达式值的多分支选择，通常适用于枚举或固定值的选择。

## 语法：

```c
switch (expression) {
    case value1:
        // 当 expression 等于 value1 时执行
        break;
    case value2:
        // 当 expression 等于 value2 时执行
        break;
    // 可以有多个 case 分支
    default:
        // 如果没有匹配任何 case，执行 default 代码
}
```
break要加，不然会导致fall-through现象
